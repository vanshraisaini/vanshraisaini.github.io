Autonomous Mobile Robot for Exploration

SLAM and AMCL


  • Pose Graph Optimization for SLAM and AMCL Localization

    Leveraging a 2D LiDAR sensor in simulation, I developed and fine-tuned a robust pose graph optimization technique for SLAM. This approach ensures accurate mapping while simultaneously utilizing AMCL (Adaptive Monte Carlo Localization) to precisely localize the mobile robot within its environment. Together, these techniques enhance the robot’s ability to navigate dynamic and unknown spaces..


  • Visual SLAM and Point Cloud Generation

    To expand the robot's capabilities, I implemented Visual SLAM using RTAB-MAP, enabling it to perceive its surroundings through a camera. This process not only allowed the robot to localize itself in real-time but also generated high-quality point cloud data for detailed 3D environmental mapping.

Tensor Board


    Reinforcement Learning for Robotic Arm Control

    In this project, I trained an RL agent for control of a robotic arm using the Proximal Policy Optimization (PPO) algorithm. The simulation environment was developed using Gazebo, where the robotic arm learned to perform tasks through iterative training. OpenAI's Baselines repository was utilized to implement the PPO algorithm, ensuring efficient and reliable training. To monitor the training progress and analyze key performance metrics, I employed TensorBoard, which provided insightful visualizations of the learning process. This project highlights my ability to integrate simulation tools, advanced machine learning algorithms, and data visualization for robotic system development.

Autonomous Weeding Robot

Autonomous Weeding Robot

    Cotton Plant and Weed detection

  • This project involved developing an autonomous robot to differentiate and eliminate weeds from cotton plants. Using YOLOv5 with transfer learning on a custom dataset,we trained a neural network model and deployed it on a Raspberry Pi 4 for real-time weed detection.
  • To enhance navigation accuracy, we fused IMU data with motor encoders for improved odometry and implemented PID control for precise movement.
  • Additionally, we designed and assembled the mechanical and electrical subsystems, creating a fully functional prototype.
